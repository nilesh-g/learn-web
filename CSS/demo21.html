CSS Properties Reference

Sr Property Name Property Description Most Common Use Cases
1 color Sets the text color of an element. Styling text, links, headings, etc.
2 background-color Sets the background color of an element. Adding background colors to sections, buttons, etc.
3 font-size Sets the size of the font. Adjusting text size for readability and design.
4 margin Sets the margin area around an element. Creating space between elements.
5 padding Sets the padding area inside an element. Adding space inside elements (e.g., buttons, containers).
6 border Sets the border of an element. Adding borders to buttons, tables, images, etc.
7 display Controls the display behavior of an element. Changing layout (e.g., block, inline, flex, grid).
8 position Specifies the positioning method for an element. Positioning elements (e.g., relative, absolute, fixed).
9 float Specifies how an element should float. Wrapping text around images or creating layouts.
10 width Sets the width of an element. Defining the size of containers, images, etc.
11 height Sets the height of an element. Defining the size of containers, images, etc.
12 text-align Aligns the text inside an element. Centering text, aligning text to the left/right.
13 box-shadow Adds shadow effects around an element. Creating depth and visual effects for cards, buttons, etc.
14 opacity Sets the transparency level of an element. Creating transparent elements or hover effects.
15 z-index Controls the stacking order of elements. Managing overlapping elements (e.g., modals, dropdowns).
16 flex Defines flexible length for flex items. Creating flexible layouts with CSS Flexbox.
17 grid Defines a grid layout for elements. Creating complex layouts with CSS Grid.
18 transition Defines smooth transitions for property changes. Adding animations for hover effects, etc.
19 transform Applies transformations like rotate, scale, etc. Creating animations and visual effects.
20 animation Defines animations for elements. Creating complex animations.
21 border-radius Rounds the corners of an element. Creating rounded buttons, images, or containers.
22 box-sizing Defines how the width and height of an element are calculated. Ensuring consistent sizing with padding and border included (e.g., border-box).
23 cursor Specifies the mouse cursor to be displayed when hovering over an element. Changing the cursor for buttons, links, or interactive elements.
24 overflow Controls how content that overflows an element's box is handled. Handling scrollable content or hiding overflow (e.g., hidden, scroll).
25 visibility Controls whether an element is visible or hidden. Hiding elements without removing them from the layout (e.g., hidden).
26 white-space Controls how whitespace inside an element is handled. Preserving line breaks and spaces in text (e.g., pre, nowrap).
27 text-decoration Adds decorations to text, such as underlines or strikethroughs. Styling links, headings, or decorative text.
28 text-transform Controls the capitalization of text. Changing text to uppercase, lowercase, or capitalized (e.g., uppercase).
29 line-height Sets the height of a line of text. Improving readability and spacing in paragraphs.
30 letter-spacing Adjusts the spacing between characters in text. Styling headings or creating visual effects in text.
31 word-spacing Adjusts the spacing between words in text. Styling paragraphs or creating visual effects in text.
32 vertical-align Aligns an element vertically within its container. Aligning text or inline elements (e.g., images) vertically.
33 background-image Sets one or more background images for an element. Adding decorative backgrounds to sections or buttons.
34 background-size Defines the size of the background image. Scaling background images to fit or cover elements.
35 background-position Sets the starting position of a background image. Positioning background images within elements.
36 background-repeat Controls how a background image is repeated. Creating patterns or preventing repetition of background images.
37 outline Draws a line around an element, outside the border. Highlighting elements (e.g., for accessibility or focus states).
38 resize Controls whether an element is resizable by the user. Allowing users to resize textareas or other elements.
39 clip-path Clips an element to a specific shape or region. Creating custom shapes or masking elements.
40 filter Applies graphical effects like blur or grayscale to an element. Creating visual effects for images or backgrounds.
