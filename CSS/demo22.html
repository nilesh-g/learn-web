CSS Display Property

The display property has several values, each with its own behavior:
  • block
  • inline
  • inline-block
  • none
  • flex
  • grid
  • inline-flex
  • inline-grid
  • table, table-row, table-cell

block elements

The element generates a block-level box. It starts on a new line and takes up the full width available.

Real-World Example: Paragraphs in a blog post or sections of a webpage.

inline elements

The element generates an inline-level box. It does not start on a new line and only takes up as much width as its content.

Real-World Example: Links in a navigation bar or text within a paragraph.

inline-block elements

The element generates a block-level box but behaves like an inline element. It does not start on a new line but can have a width, height, padding, and margin.

Real-World Example: Buttons in a toolbar or icons in a navigation menu.
