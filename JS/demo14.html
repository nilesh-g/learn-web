In browser DOM, the window and document objects are fundamental, with window representing the browser window and document representing the HTML document itself, allowing JavaScript to interact with and manipulate the page's content and structure.

window object:
  • Represents the browser window and its contents.
  • Provides access to browser features like size, history, and location.
  • All global JavaScript objects, functions, and variables become members of the window object.
  • In <script>, "this" keyword refers to window object.
document object:
  • Represents the HTML document loaded in the browser window.
  • Is the root node of the HTML document, allowing access to all elements within it.
  • Provides methods and properties to access, modify, and manipulate the document's structure, content and decoration.
  • Methods to access various elements:
    1. getElementById()
    2. getElementsByTagName()
    3. getElementsByName()
    4. getElementsByClassName()
    5. querySelector() -- select elements by given CSS selector.
